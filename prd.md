\# Ürün Gereksinim Dokümanı (PRD)



\## Temel Özellikler

1\. \*\*Basitleştirici Girişi:\*\* Kullanıcının karmaşık metni yapıştırabileceği geniş bir alan.

2\. \*\*AI İşleme:\*\* Gemini API ile metni "Adım 1, Adım 2..." formatına indirgeme.

3\. \*\*Erişilebilirlik:\*\* Minimum 20px font, yüksek kontrastlı renkler (Sarı/Siyah).

4\. \*\*Sesli Okuma:\*\* Basitleştirilen metni cihazın ses sentezleyici ile seslendirme.



\## Teknik Gereksinimler

\- React / Tailwind CSS

\- Gemini 1.5 Flash API (Hız için)

\- Vercel/Lovable Deployment
